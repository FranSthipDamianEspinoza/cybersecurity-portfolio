# Lab 01 - Evaluación Básica de Seguridad Web
## Descripción general

Este laboratorio documenta una evaluación básica de seguridad realizada sobre un sitio web con autorización de su propietario y con fines exclusivamente educativos.
El objetivo principal fue aplicar las herramientas y metodologías aprendidas durante el módulo de **Fundamentos Técnicos Avanzados para Red Team** y el módulo de **OSINT y Gestión Profesional de Vulnerabilidades**, documentando todo el proceso como parte de un portafolio profesional de ciberseguridad.

## Objetivos del laboratorio

Durante esta práctica se buscó:
- Comprender la estructura funcional de una aplicación web real.
- Realizar reconocimiento pasivo y activo.
- Identificar la superficie de ataque expuesta.
- Enumerar servicios y recursos públicos.
- Analizar solicitudes HTTP mediante Burp Suite.
- Documentar observaciones y posibles oportunidades de mejora desde el punto de vista de la seguridad.

## Alcance

La evaluación se realizó únicamente sobre recursos autorizados por el propietario del sitio.
Las actividades se limitaron a:
- Reconocimiento
- OSINT
- Enumeración
- Inspección de tráfico HTTP
- Revisión manual de funcionalidades

No se realizaron actividades de explotación no autorizada ni modificaciones sobre el sistema.

## Herramientas utilizadas

Durante el laboratorio se emplearon las siguientes herramientas:
- Nmap
- Gobuster
- Burp Suite Community
- Navegador web
- Google Search (OSINT)
- Kali Linux

## Aviso

Este laboratorio tiene fines exclusivamente educativos y forma parte de un portafolio personal de ciberseguridad.
Todas las actividades fueron realizadas con autorización del propietario del sitio y dentro del alcance previamente acordado.
